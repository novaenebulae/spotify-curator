# 13 — Guide de collaboration avec Cursor

Tags: #cursor #workflow #prompting

## Démarrage session

Demander à Cursor :

```text
Lis AGENTS.md, docs/00-project-brief.md, docs/01-architecture.md,
docs/03-development-workflow.md et backlog/phase-0.md.
Ne code rien. Résume le périmètre, les décisions et les risques.
```

## Format de demande

```text
Phase : X
Tâche : X.Y
Contexte : ...
Critères d’acceptation : ...
Contraintes : ...
Tests attendus : ...
Instruction : propose un plan avant de coder.
```

## À éviter

- « Développe tout le projet »
- « Implémente toutes les phases »
- « Ajoute Spotify, ReccoBeats, yt-dlp, Essentia, clustering et UI complète »

## Bon rythme

- une tâche ;
- peu de fichiers ;
- tests ;
- revue ;
- documentation.
